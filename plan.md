# 閱讀筆記系統 - MVP 開發計劃 v2.0

## 🎯 專案定位
一個**行動優先**的個人閱讀筆記管理工具，幫助讀者隨時隨地記錄閱讀心得，建立個人知識體系。

## 📊 現狀分析

### 已完成功能（2025-10-21）
✅ 用戶認證系統（註冊/登入）  
✅ 書籍CRUD操作（新增/查看/編輯）  
✅ 三維筆記系統（摘錄/思考/實踐）  
✅ 基礎搜尋與標籤篩選  
✅ 跨書籍筆記瀏覽（主題整理）  
✅ Markdown 筆記支援  
✅ 筆記自動儲存（30秒）  
✅ Markdown 工具列  
✅ 鍵盤快捷鍵（Ctrl+S, Esc）  
✅ 書籍封面上傳  
✅ 筆記圖片上傳  
✅ 即時搜尋與高亮  
✅ 載入骨架屏  
✅ 響應式設計（基礎）  

### 核心價值
- **行動優先**：任何裝置隨時記錄
- **結構化筆記**：摘錄→思考→實踐的知識轉化
- **快速檢索**：即時搜尋與標籤篩選
- **數據安全**：自動儲存，永不遺失

## 🛠️ 技術棧

### 前端
- **核心**: React 18 + TypeScript + Vite
- **樣式**: Tailwind CSS + shadcn/ui
- **狀態**: React Query + Context
- **測試**: Vitest + Testing Library + Playwright
- **PWA**: Vite PWA Plugin

### 後端
- **BaaS**: Supabase（認證、資料庫、Storage）
- **資料庫**: PostgreSQL + RLS
- **檔案**: Supabase Storage (book-covers, note-images)

### 工具鏈
- **程式碼品質**: ESLint + Prettier
- **測試**: Vitest + Playwright
- **CI/CD**: GitHub Actions
- **效能**: Lighthouse CI

## 📝 資料庫結構

### books 表
```sql
- id: UUID PRIMARY KEY
- user_id: UUID REFERENCES auth.users
- title: TEXT NOT NULL
- author: TEXT NOT NULL
- tags: TEXT[]
- progress: ENUM('未讀', '閱讀中', '已讀')
- rating: INTEGER (1-5)
- cover_url: TEXT
- created_at: TIMESTAMPTZ
- updated_at: TIMESTAMPTZ
```

### notes 表
```sql
- id: UUID PRIMARY KEY
- book_id: UUID REFERENCES books
- section: ENUM('摘錄', '思考', '實踐')
- content: TEXT
- created_at: TIMESTAMPTZ
- updated_at: TIMESTAMPTZ
```

### note_images 表
```sql
- id: UUID PRIMARY KEY
- note_id: UUID REFERENCES notes
- image_url: TEXT NOT NULL
- created_at: TIMESTAMPTZ
```
